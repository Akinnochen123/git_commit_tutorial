A BRIEF TUTORIAL ON HOW TO USE GIT COMMIT

We will be seeing how to use the git command "git commit" in simple steps in this tutorial

The following are sequencial steps to follow to use git commit

  • STEP 1: Install git and create a GitHub account

    Follow the instructions here to install git (if not already installed)

    Once you've done that, create a GitHub account here

  • STEP 2: Create a local git repository

    To use git we'll be using the terminal also known as command prompt, but here we will be using our newly installed git bash

    Open up git bash and move to where you want to place the project on your local machine using the "cd" (change directory) command e.g cd Desktop to place your project on your desktop.

    After you cd into your preferred location, create a new folder by using the "mkdir" (make directory) command e.g mkdir project_folder

    After creating the folder, initialize a git repository in the root of the folder by using the "git init" command

  • STEP 3: Add a new file to the repository

    Go ahead and add a new file using the "touch" command e.g touch new_file

    After creating the file you can use the "git status" command to see which file git knows exist

  • STEP 4: Add a file to the staging area

    Add your file to the staging area by using the "git add" command

    Rerun the "git status" command and you'll see that git has added the file to the staging area (notice the 'changes to be committed'line)

    To reiterate, the file has not yet been added to a commit, but it's about to be

  • STEP 5: Create a commit

    It's time to create your first commit!

    Run the command "git commit -m" 'your message about the commit' e.g git commit-m "first commit"

    You might be wondering what the -m is for, the -m is a flag, I'll discuss more on this in the latter part of the tutorial

    Run the command "git log" to see your commited changes

    Yaay! you've just done your first commit

    Practice this steps more times to become a pro

Flags that are commonly used with git commit

-a flag: it is used to add all your files to your commit

-m flag: it is used to connect a commit message to your commit

-am flag: it is a combination of the a and m flags and it is used to add and create a message for the commit in one command
